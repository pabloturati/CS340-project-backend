# CS340-project-backend

This repository holds the backend for Oregon State CS340-Databases

# Scripts to start the project

## Development modes

### To run development with debugging

`npm run dev`

### To run development on a windows command

`npm run dev-windows`

### To start server

`npm start`

## Using Forever

### To start forever

`npm start-forever`

### To stop forever

`npm stop-forever`
